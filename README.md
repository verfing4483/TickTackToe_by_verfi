# 🎮 Руководство пользователя: Игра «Крестики-Нолики 1vs1»
## 👥 Часть 1. Руководство для игроков
### 1. Начало игры

•	Игра рассчитана строго на двух игроков за одним устройством.

•	При запуске игры на экране появится надпись: «ХОД ИГРОКА (КРЕСТИКИ)».

•	Игрок 1 всегда ходит первым и управляет Крестиками. Игрок 2 управляет Ноликами.

### 2. Игровой процесс и управление

•	Игровое поле состоит из 9 клеток (сетка 3 X 3).

•	Игроки ходят по очереди, нажимая левой кнопкой мыши на любую свободную клетку.

•	Если клетка уже занята, повторно нажать на неё нельзя — ход не засчитается.

•	Информационный текст сбоку экрана всегда подсказывает, чья сейчас очередь: «ХОД ИГРОКА (КРЕСТИКИ)» или «ХОД ИГРОКА (НОЛИКИ)».

### 3. Условия победы и окончания игры

•	Победа: Побеждает тот игрок, который первым соберет линию из 3-х своих фигур по горизонтали, вертикали или диагонали.

o	При победе Крестиков на экране появится надпись «ПОБЕДИЛ ИГРОК (КРЕСТИКИ)!», а выигрышная комбинация перечеркнется красной линией.

o	При победе Ноликов появится надпись «ПОБЕДИЛ ИГРОК (НОЛИКИ)!», а линия будет синей.

•	Ничья: Если все 9 клеток заполнены, но никто не собрал линию из трех фигур, игра объявляет «НИЧЬЯ!».

### 4. Перезапуск игры

•	Как только игра завершилась (победой или ничьей), она мгновенно начнётся заново.

## 🛠️ Часть 2. Руководство для разработчика 

Если вы переносите этот код в Unity, вам необходимо правильно настроить компоненты на сцене. Скрипт Game должен быть прикреплен к любому управляющему объекту (например, [GameManager]).

### 🗂️ Настройка инспектора 

Перетащите соответствующие объекты со сцены в поля скрипта:

1.	Crosses (Массив GameObject, 9 элементов): Сюда нужно по порядку (от 0 до 8) перетащить графические объекты крестиков для каждой из 9 клеток поля.

2.	Zeros (Массив GameObject, 9 элементов): Сюда по порядку (от 0 до 8) перетаскиваются графические объекты ноликов.

3.	Redline / Blueline (GameObject): Спрайты или UI-линии красного и синего цвета. Изначально на сцене они могут быть включены — скрипт сам спрячет их при старте (SetActive(false)).

4.	Info Text (Text): Компонент UI Text (Legacy), в который будет выводиться состояние игры и результаты.

### ⚠️ Важно: Массивы crosses и zeros должны заполняться строго в соответствии с индексами клеток игрового поля (слева направо  , сверху вниз):

•	0, 1, 2 — Верхний ряд

•	3, 4, 5 — Средний ряд

•	6, 7, 8 — Нижний ряд

### 📐 Как работает логика линий победы

Скрипт автоматически рассчитывает, где и под каким углом включить линию победы, используя координаты центральных клеток:

•	Горизонтали: Центрируются по позициям крестиков из среднего столбца (индексы 1, 4, 7), угол поворота 0

•	Вертикали: Центрируются по среднему ряду (индексы 3, 4, 5), разворачиваются на 90

•	Диагонали: Центрируются строго в центральной клетке (индекс 4), разворачиваются на    -45 и +45
